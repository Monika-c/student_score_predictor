# student_score_predictor
Machine Learning project that predicts student marks based on study hours using Linear Regression. Includes data analysis, visualization, model training, and performance evaluation.
📌 Project Overview

This project builds a Machine Learning model to predict student marks based on the number of study hours.

The goal is to help an EdTech platform estimate student performance in advance and identify students who may need additional academic support before exams.

📊 Dataset Information
The dataset contains:
Hours → Number of hours studied
Marks → Score obtained by the student
The dataset shows a positive relationship between study hours and marks.

🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Jupyter Notebook

📈 Project Workflow
Data Loading
Data Exploration
Data Visualization (Scatter Plot & Trend Line)
Train-Test Split
Model Training using Linear Regression
Model Evaluation (R² Score & MSE)
Final Prediction

🤖 Model Used
Linear Regression
Linear Regression was chosen because the relationship between study hours and marks is linear.
